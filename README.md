# Group_Dna-
A python based whatsapp chat analyzer

Group_Dna is a Python-based chat analyzer that converts exported WhatsApp chat files into comprehensive personality and activity reports. 📱💬
🚀 Overview
Ever wondered what your group chat really says about you? This project parses raw chat data to uncover behavioral patterns, response times, and personality archetypes among group members. It’s a great tool for turning unstructured text data into fun, actionable insights.
🛠️ Features
Activity Heatmaps: Visualize when participants are most active throughout the day.
Response Analytics: Automatically calculate the fastest and slowest repliers in the group.
Personality Archetypes: Identifies roles like "The Spammer," "The Drama Queen," "The Storyteller," "The Night Owl," and more.
Word Frequency: Analyzes favorite vocabulary while filtering out common stop words.
Group Overview: Generates statistics on chat duration, message volume, and participation percentage.
⚙️ Technical Stack
Language: Python
Libraries: NumPy (for data matrices), datetime (for time-series analysis)
Processing: Custom parsing logic for unstructured .txt exports.
📊 How to Use
Export your WhatsApp chat as a .txt file.
Place your exported file (e.g., hostel_bois.txt) in the same directory as the project.
Run the Anona_GroupDna.ipynb file.
View the generated report directly in your console or notebook output.
